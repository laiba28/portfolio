Web Automation

Snapchat Web Messaging Automation System

A desktop-based automation system for managing Snapchat Web conversations, supporting both manual messaging and AI-powered automated replies.

Project Overview

This project is a web-based messaging automation system built on top of Snapchat Web. It allows users to send messages manually to specific usernames or automatically manage incoming conversations at scale. All message activity is persisted for tracking, analysis, and follow-ups.

Core Capabilities

  • Manual message sending to specific Snapchat usernames
  • Automated scanning of full DM conversation lists
  • Detection of latest incoming messages
  • AI-generated replies using OpenAI
  • Conversation history persistence
  • Desktop-level automation via Electron

Manual Messaging Flow

In manual mode, the user enters a Snapchat username and a message through the system interface. The automation layer launches Snapchat Web, navigates to the target user’s chat, sends the message, and stores the message details in the database under the conversations table.

Automated Messaging Flow

In automated mode, the system opens the full DM list and iterates through each conversation. It inspects the latest message to determine whether it was sent by the other user. If so, the full chat context is saved, passed to the AI engine, and an appropriate response is generated and sent automatically on Snapchat Web.

Data Storage & Tracking

All conversation data, including sender, message content, timestamps, and reply status, is stored in a structured database. This enables historical tracking, response auditing, and future message analysis.

Technology Stack

Python Electron.js Puppeteer Snapchat Web OpenAI APIs Database Persistence

System Architecture

The desktop application serves as the orchestration layer, managing browser sessions and automation logic. Puppeteer handles interaction with Snapchat Web, while the Python backend coordinates workflows, AI responses, and database operations. This separation allows for scalable messaging automation without relying on unofficial APIs.

Reliability & Safeguards

The system includes checks to avoid duplicate replies, ensures that only inbound messages trigger automated responses, and gracefully handles session timeouts or UI changes in Snapchat Web.
